# 茶馆 H5 单页介绍

## 项目简介

本项目是一个面向移动端的茶馆服务介绍 H5 单页应用，适配手机浏览器（320px～414px 宽度），提供茶馆基本信息、服务项目、团购套餐、门店介绍及预定电话、一键导航等功能。点击预定电话，直接调起手机拨打电话功能，点击一键导航，直接调起手机上安装的地图app。

## 功能模块

### 1. 顶部导航栏
- 返回按钮、分享、收藏操作
- 门店名称展示

### 2. Banner 轮播图
- 多张门店实景图片轮播
- 数字角标

### 3. 门店基本信息
- 门店名称展示
- 服务标签（棋牌·茶室·商务·24小时·免费WiFi·可自助）
- 营业时间
- 地址 + 导航
- 电话联系

### 4. 预订 CTA
- 突出"立即预订"按钮，引导用户转化

### 5. 优惠套餐（团购套餐）
- 新品套餐卡片列表
- 套餐名称、价格、优惠标签、半年销量
- 抢购按钮

### 6. 门店介绍
- 文字简介（环境、设备、特色服务等）
- 开业福利活动说明

### 7. 底部 Tab 导航
- 首页 / 会员专属 / 我的预订 / 个人中心

## 技术栈

| 类型 | 技术 |
|------|------|
| 结构 | HTML5 |
| 样式 | CSS3（Flex / Grid / 变量 / 动画） |
| 交互 | jQuery 3.x + 原生 JS |
| 字体图标 | Font Awesome 6 |
| 轮播 | 纯 CSS + JS 实现（无额外依赖） |

## 设计规范

- **主色调**：暖金棕 `#8B6914`、浅金 `#C8A45A`
- **背景色**：奶油白 `#FDF8F0`
- **强调色**：橙红 `#E8522A`（CTA 按钮）
- **字体**：系统默认中文字体栈
- **基准宽度**：375px（iPhone 标准），最大宽度 480px 居中展示
- **图片**：使用渐变色占位，上线时替换为实际图片

## 使用方式

直接用浏览器打开 `index.html` 即可预览；建议用 Chrome 开发者工具切换至手机视图（375×812）查看最佳效果。
